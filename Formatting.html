Headings

Headings determine the hierarchy of your website content. You need an opening and closing tag on elements. The main topic of your page should eb your H1. Then the subtopics, should be H2's. H1 is a bit larger than H2. We can use headings all the up to H6's. In reallity, how headings look on your page have little do with their purpose as this actually has to do with the structure of your page only.

Paragrpahs

Paragraphs need opening and closing tags within your site as well. It is very easy to forget about your closing tag on paragraphs. This tag should be used for every single indivudal paragraphs. HTMl gives automatic spacing between elements.

Line Breaks

If you need to create space between two iteams, but the two items are still related such as an address. We will not separate them into seperate them into different paragraphs. We will use the line break tag(lefft angle bracket)br(right angle bracket). You want to use this tag when you want a hard return without creating a new paragraph.

Emphasizing Text

There are four main tags that we use to empahsis text they are

  • bold
  • strong
  • Italic
  • Emphasis


Bold Tag

To use the bold tag, we will use a (left angle bracket)b(right angle bracket). You need to be careful as whatever you wrap within this tag will get bolded.This will separate this text from the rest of it.

Strong Tag

The strong tag is shown by (left angle bracket)strong(right angle bracket). When you use this tag, you are showing a text as having strong importance. It is typically bolded text that has this tag as well.

Italic Tag

The italic tag is string of text that is set apart stylistically from the text around it. This is helpful on phrases or terms that should stand out.

Emphasis Tag

The emphasis tag is used to denote text that is emphasized more than the text aorund it. It is usually shown in italics. It should NOT be used in place of the italics tag.

Controlling WhiteSpace

If we want to add whitespace, we need to use a charcter known as a non breaking space. We really want to use CSS for controlling the amount of whitespace on our pages. It is really used if you need two words to stay together when someone is altering the size of your page.
