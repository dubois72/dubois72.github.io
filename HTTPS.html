The reasons for using HTTPS

HTTPS stands for hyper-text transfer protocol secure. HTTP stands for hyper-text transfer protocol. HTTPS is the more secure option when you are deciding what you want to do with your site.

Every site should use HTTPS, not just sites where you will have things like credit card infomration. Any time you are collecting information from people you should be using HTTPS. As a user, you should not trust sites that do not use HTTPS as it is easy for hackers to access your information.

There is many reasons for using HTTPS which inluce:

  1. Speed Boost
  2. Higher Search Engine Rankings
  3. Fewer Browser Warnings
  4. Improved Security
