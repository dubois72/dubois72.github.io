DevTools

Chrome Devtools is a resource that you will use as a beginning a devloper and throughout your entire career even as an extermely experienced developer. As technology is always changing, we as devlopers must continually evolve or we risk falling behind with the times.The first step to accessing devtools is to download google chrome as google is the provider of this resource. The way that we access chrome devtools is by selecting the chrome menu which is symbolized by three dots. From there, you will go down to tools and then select Developer Tools.There are shortcuts to open your Devtools for windows the shortcut is control+shift+I and for Mac it is Cmd+Opt+I. These shortcuts are helpfil in speeding up your productivity time.

Inspect Tool of DevTools

The first tool that you should learn is to to insepect an element. Once we have opened to DevTools, we can right-click on any content on the page and that will inspect the element that we are looking at. This will give you an idea of what the code of that element looks like. As well you can change this code within your broser which will show you what the site would look like if you implemented your own code.

The 8 Main Groups of Devtools

The 8 elements involved with devtools are elements, resources, network, sources, timeline, profiles, audits, and the console. Elements are what we covered earlier were you can inspect specfic parts of pages.The resources tool allows you to view and edit local and sessions storage. This has mainly to do with databases.The network tool allows you to record and anaylze network activity as well as load information for the page. The sources tool allows you to view files and edit the CSS and JavaScript of the website. The timline tool allows you to record and analyze all the activity in your application as it runs. The profiles tool allows you to profile the execution time and memory usage of a web app or page. The audits tool allows you to run network utilization and performance audits on your web app. The console tool is used to debug your JavaScript.
